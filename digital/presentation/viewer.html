Presentation Design System

AI がスライドデッキ（HTML）を生成する際に従うルールセットのビジュアルリファレンスです。16:9 (1920×1080) ダークファーストスライド。

01 — 基本設定

スライドサイズ: 1920×1080px（16:9）
キーボード操作: ←→ / スペース でスライド切り替え
フォント: DM Sans (300,400,500,600) + JetBrains Mono (400,500) 

body {
  font-family: 'DM Sans', -apple-system, BlinkMacSystemFont, sans-serif;
  background: #000000;
  color: #e8e8ec;
  overflow: hidden;
  width: 100vw; height: 100vh;
}

04 — マテリアル

Glass Card
基本的なガラスカード。半透明背景 + blur エフェクト。
Liquid Glass（強調カード）
グラデーション背景 + saturation + brightness でリッチな質感。

Glass Card CSS

background: rgba(10, 10, 12, 0.6);
backdrop-filter: blur(24px); -webkit-backdrop-filter: blur(24px);
border: 1px solid rgba(255, 255, 255, 0.06);
border-radius: 8px;
padding: 48px 40px;

07 — キーボード操作

←→ または スペースキー でスライドを切り替えます。 

document.addEventListener('keydown', function(e) {
  if (e.key === 'ArrowRight' || e.key === ' ') { nextSlide(); }
  else if (e.key === 'ArrowLeft') { prevSlide(); }
});

08 — 禁止事項

  • 本文を 16px 未満にしない
  • font-weight: 700 は使わない
  • 1スライドにカード 3枚 or 箇条書き 5項目が上限目安
  • 背景色を #000000 以外にしない
  • スライドの padding を 64px 80px 未満にしない
