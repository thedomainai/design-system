Paper Presentation Design System

AI がスライドデッキ（HTML）を生成する際に従うルールセットのビジュアルリファレンスです。16:9 (1920×1080) モノクロームスライド。

01 — 基本設定

スライドサイズ: 1920×1080px（16:9）
キーボード操作: ←→ / スペース でスライド切り替え
フォント: Crimson Pro (300,400,600) + Instrument Sans (400,600,700) + JetBrains Mono (400,500) 

body {
  font-family: 'Instrument Sans', -apple-system, BlinkMacSystemFont, sans-serif;
  background: #fdfdfc;
  color: #1a1a1a;
  overflow: hidden;
  width: 100vw; height: 100vh;
}

04 — マテリアル

Flat Paper
基本的な紙カード。角丸なし、シャドウなし。
Layered Paper（強調カード）
オフセットシャドウで重なり感を表現。角丸なし。

Flat Paper CSS

background: #faf9f7;
border: 1px solid #e0e0e0;
border-radius: 0;
padding: 48px 40px;

Layered Paper CSS

background: #fdfdfc;
border: 1px solid #c4c4c4;
border-radius: 0;
box-shadow: 2px 2px 0 #f0f0f0;

Editorial Rule（セクション区切り）

width: 100%; height: 2px;
background: #000000;
margin: 48px 0;

Double Line（重要区切り）

width: 100%; height: 0;
border-top: 3px double #000000;
margin: 48px 0;

07 — 禁止事項

  • 本文を 16px 未満にしない
  • カラー（彩色）を使用しない（グレースケールのみ）
  • カードに角丸をつけない（radius-none: 0px）
  • border-radius を radius-md（4px）を超えて使用しない（50% の円形を除く）
  • グラデーション・グロウ・ブラーなどの装飾エフェクトを使用しない
  • 1スライドにカード 3枚 or 箇条書き 5項目が上限目安
  • 背景色を #fdfdfc 以外にしない
  • スライドの padding を 64px 80px 未満にしない
